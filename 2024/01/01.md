## わかったこと
- console.log();でコンソールに（）で囲んだ値を出力する。
- 変数はlet 変数名 = 値;で定義する。
- 定数...const 変数名 = 値;で定義する、変数と違い値を変数 = 更新したい値;で更新することができない。
- テンプレートリテラル、Rubyの変数展開のように文字列の中に変数、定数を書くことができる`${変数(or 定数)}`
- while文...while(条件)で条件の真偽値が真である時、繰り返し処理を実行する。条件を抜け出すためのコードを書かないと無限ループになってしまうので注意する。
- for文...for (変数の定義;条件;変数の更新) {処理}で書く。
- 配列...let or const 配列名 = [値、値、値]；で定義する。配列は、定数で定義した場合は要素の値を更新することができるが、再代入することはできない
- ex)example[0] = "sample"; 配列の要素の更新はできるが const example = ["sample"];は定数に再代入となりエラー。
- オブジェクト...[プロパティ名: 値, プロパティ名:　値];とすることで定義できる。プロパティ名には文字列を用いる。
- 配列と組み合わせることができ、オブジェクトを要素に持つ配列のような使い方ができる。
- 配列もオブジェクトも存在しないインデックス番号やプロパティの要素を取得しようとするとundefinedという特別な値が出力される。
- オブジェクトの値には文字列や数値だけではなく、オブジェクトも使用できる。
